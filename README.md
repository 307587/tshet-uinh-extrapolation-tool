# Qieyun Extrapolation Tools

This is the Japanese Kan-on version. For the Cantonese Version, see the `main` branch.

## Build

```sh
./prepare.sh
python build.py
```

## View

Open `qieyun.sqlite3` with DB Browser for SQLite, then load `extrapolate.sql`.
